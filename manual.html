⚙️ 1. Installation & First Steps

Setting up your scripting workstation takes less than two minutes. Follow the procedures outlined below to configure your user signature and project parameters.

1. Running the Setup

Download the installer bundle ReelScript_Setup.exe and launch it on your Windows machine. The wizard registers custom file systems and links all .rsp (ReelScript Project) files to the workspace automatically. A desktop shortcut is added for easy loading.

2. Author Profiles (Crucial step)

ReelScript utilizes a custom tagging signature to attribute code revisions. Before writing, please define your editor credentials:

  • Launch the ReelScript application window.
  • Navigate to the top menu bar: Customize → Author Profile & Revision Color.
  • Provide your signature/name and click the swatch input to designate a unique revision mark color (e.g. Purple, Green, Blue).
  • Click Save Profile to set the signature.

🤝 2. Collaboration & Revisions

Working in teams on a single script can often lead to accidental edits or missing context. ReelScript solves this through a dedicated Revision Tracking engine that visualizes modifications inline.

Did You Know?

ReelScript project files are consolidated into compressed .rsp bundles containing metadata, scripts, snapshots, and revision histories inside a single structured file.

⏱️

Real-time Marking

Any edits made while revision tracking is active will place a colored asterisk (*) on the left margin, signaling changes immediately.

💬

Revision Notes

Right-click any revision asterisk to attach contextual notes. This opens the Revision Document and keeps comments synced.

✅

Approve & Merge

Review team edits inline. Right-clicking an asterisk allows you to Accept and Merge edits to remove marks and make changes permanent.

Figure 2.1: Revision and collaboration workflow matching edits with distinct author tokens.

Sharing Edits Safely

ReelScript includes direct shortcuts to synchronize and exchange scripts across distributed storage vaults:

  1. Save your ongoing project draft: Navigate to File → Save Project (Ctrl+S).
  2. Click Share → Open Shared Google Folder from the menu toolbar.
  3. The system opens your configured Google Drive or cloud sync folder directly in your browser.
  4. Drop the updated .rsp file in the target directory to sync your edits with co-writers.

🎬 3. The Interface & Writing Flow

Writing in screenplay format is highly structured. ReelScript optimizes this formatting step by automating layout shifts based on native screenplay logic.

1. Navigation Sidebar

The workspace sidebar features three primary utility tabs:

  • 📋 Script Navigation: Navigate through documents like your Title Page, Screenplay Draft, and Revision Notes. Right-click any document to Move Up, Move Down, Rename, or Delete.
  • 🎬 Scenes Explorer: A live directory listing scene headings. Clicking elements scrolls you to that point instantly.
  • 💬 Private Notes: Local-only scratchpad. Any thoughts entered here will stay on your local disk and will not sync inside the shared project file.
  • 💾 Save: Quick access button to instantly save your project.
  • 📌 Stay On Top: Toggles the window to float above all other applications. The pin icon turns red when active.

2. Smart Formatting Flow

Write content without manual formatting buttons. The editor automatically adjusts margins and letter casings based on keyboard inputs:

  • Type a Scene Heading & press Enter → Switch to Action.
  • In Action, press Tab → Switch to Character name.
  • Type a Character & press Enter → Switch to Dialogue box.
  • In Dialogue, press Enter → Switch back to Character block.
  • Pressing Enter on a blank Action line automatically converts it to a Character element.

💽 5. Backups & Auto-Save (Never lose work)

A screenwriter's worst nightmare is losing drafts. ReelScript mitigates this by maintaining a redundant, multi-tier backup system.

Figure 5.1: Multi-tiered file synchronization and automatic backup architecture.

Backup Customization

Press Ctrl + Shift + S (or select File → Manage Backups & Auto-Save) to modify directory and storage configs:

  1. Save Interval: Adjust background backup timings (default: 5 minutes). Additionally, ReelScript initiates an instantaneous local autosave whenever typing pauses for 1 second.
  2. Local Directory: By default, files are routed to your local machine path: Documents/ReelScript.
  3. Secondary Cloud Mirror: Route duplicate backup files to synced Google Drive, OneDrive, or Dropbox folders for redundancy.
  4. Backup Retention Limits: To save disk space, directories are limited to 5 backups. The 6th file triggers the removal of the oldest copy.
Manual Snapshots

Before performing major section rewrites, click Revisions → Manage Snapshots to create a permanent snapshot state. This locks in your current draft so you can roll back to it if a rewrite goes off the rails.

🛠️ 6. Useful Tools & Features

ReelScript includes workflow accelerators to clean up scripts, review character dialogue, and export files to other production systems.

👥

Character Filter

Select Tools → Character Filter to isolate dialog boxes for a single role. Perfect for assessing character tone and flow.

✏️

Global Rename

Select Tools → Rename Character to modify a character name across both dialogue lines and action scene details.

🔍

Find & Replace

Press Ctrl + F to launch search panels. Match exact uppercase characters or replace instances globally.

📖

Spellcheck & Dict

Select Tools → Check Spelling. Ignores screenwriting shorthand like EXT. and INT., and lets you add custom terms to the dictionary.

🕶️

Focus & Dark Mode

Press F11 to toggle Focus Mode, hiding toolbars. Toggle Customize → Dark Mode to switch themes and ease eye strain.

📤

Industry Exports

Export draft pages to standard formats like PDF or native .fdx (Final Draft) format, ensuring seamless sharing with producers.
